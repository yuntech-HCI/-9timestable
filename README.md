# -9TimesTable  
學齡孩童九九乘法（2×1-9×9）互動練習遊戲

## 📌 專案簡介  
「-9TimesTable」是一款適合小學兒童使用的互動學習遊戲，透過按鍵（或點擊螢幕）輸入答案、即時回饋答對／答錯、錯題優先練習，幫助孩子熟練九九乘法。遊戲採 WebGL／Unity 架構，可於桌機、平板、手機瀏覽器中使用。

## 🎮 核心功能  
- 題庫涵蓋 2×1 至 9×9 的所有題目（共 72 題）  
- 點擊畫面或鍵盤數字鍵即可開始遊戲  
- 錯題優先模式：答錯的題目會優先再出，直到全部題目皆正確為止  
- 按鍵動畫反饋（點擊數字鍵盤或螢幕按鈕）  
- 自動偵測螢幕方向（直式／橫式）並調整佈局  
- 簡單、直覺、適合兒童操作的 UI  

## 📱 支援裝置  
- 桌機瀏覽器（Windows、macOS）  
- 平板／手機瀏覽器（iOS、Android）  
- WebGL 遊戲形式，適合教學、校內導入、家庭練習  

## 🚀 使用方式  
1. 透過瀏覽器開啟遊戲：  
   [遊戲網址](https://yuntech-hci.github.io/-9timestable/)  
2. 點擊畫面任意位置或按下鍵盤數字鍵開始遊戲  
3. 題目出現在畫面上，輸入答案後按 Enter 或點擊「e」鍵確認  
4. 若輸入錯誤，該題會加入錯題庫，稍後優先再出現  
5. 正確答案自動進入下一題，錯題重複練習直到掌握  

## 🛠 專案結構  
- `Assets/`：Unity 專案資源  
- `Scenes/`：主場景 (main)  
- `Scripts/main.cs`：遊戲主腳本，包含題庫、輸入邏輯、錯題優先邏輯、UI 排版  
- `Build/WebGL/`：輸出至 GitHub Pages 的 WebGL 建構檔  

## ✏️ 教育用途
